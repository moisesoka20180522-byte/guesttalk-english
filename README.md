# 민박 영어 회화 트레이너

## PC에서 열기

`index.html`을 브라우저로 열면 됩니다.

## 스마트폰에서 열기

1. PC와 스마트폰을 같은 Wi-Fi에 연결합니다.
2. `start-phone-server.bat`을 실행합니다.
3. 창에 표시되는 `Phone:` 주소를 스마트폰 브라우저에 입력합니다.

예시:

```text
Phone: http://10.18.198.68:18765
```

사용 중에는 실행 창을 닫지 마세요. 끝낼 때는 `Ctrl+C`를 누릅니다.

## 열리지 않을 때

- 스마트폰이 PC와 같은 Wi-Fi인지 확인합니다.
- Windows 방화벽 확인 창이 뜨면 Node.js 접근을 허용합니다.
- 공용 Wi-Fi는 기기끼리 접속을 막는 경우가 있습니다. 그때는 휴대폰 핫스팟에 PC를 연결한 뒤 다시 실행합니다.
